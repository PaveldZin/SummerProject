Flask приложение, ищущее аниме по скриншоту и выводящее основную информацию.
Для поиска используется trace.moe-api, принимающий ссылку на изображение или сам скриншот.
Если источник найден, из ответа получаем id на сайте anilist.co.
По id c помощью AniList Api v2 получаем нужные данные и выводим их.
Для запуска используется main.py
Форма находится по адресу http://localhost:5000/form
